minfree :: [Int] -> Int Source #

与えられた自然数のリストに含まれない最小の自然数を求める関数

自然数は0を含む

前提条件1: 与えられたリストには順序がついていない

前提条件2: 要素は重複していない

>>> minfree [8, 23, 9, 0, 12, 11, 1, 10, 13, 7, 41, 4, 14, 21, 5, 17, 3, 19, 2, 6]
15
